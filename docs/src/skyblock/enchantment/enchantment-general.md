# 空岛附魔指南

:::info
我们的空岛为各种武器和工具添加了近千种新的附魔，每种附魔都有独特的效果，你可以在[这里](./database/index.md)检索相应的附魔。
:::  

这篇文章向你介绍服务器中有关附魔方面的功能和他们的用法。  

## 附魔大全  

这个功能可以让你在游戏中检索你需要的附魔。  

## 附魔查询

单击背包中你想要查询的物品，在左下角选择对应的查询模式进行查询。

## 铁砧模拟器

服务器解除了原版中铁砧过于昂贵的机制，现在虽然铁砧会显示过于昂贵，但是只要你有足够的经验，你仍然可以附魔。你可以通过使用铁砧模拟器来查看附魔需要多少级的经验。  
假如你无法进行拼合，铁砧模拟器还会告诉你无法拼合的原因。
